subzero-paper
=============

Start in `main.tex`

  make
  make cameraready


Required
===========


pdftops

  # on Mac OSX with brew
  brew install libtiff
  ln -s /usr/local/Cellar/libtiff/*/lib/libtiff.dylib /usr/local/lib/libtiff.3.dylib
  brew install xpdf


ps2pdf14

  # install ghost script
  brew install gs
